## 📊 Presentación interactiva (HTML)

Abre `index.html` en cualquier navegador moderno para visualizar la presentación completa con navegación por teclado:

- `→` / `←` — Cambiar de slide
- `Home` / `End` — Primera / última slide
- `F` — Pantalla completa
- Click en los puntos inferiores para saltar a una slide específica

## 🛠️ Stack del software

**Frontend:** React 18 · TypeScript · Vite · TailwindCSS · Zustand · React Hook Form · Zod · Axios · Lucide

**Backend:** Node.js · Express · Prisma · PostgreSQL (Neon) · JWT

**Testing:** Vitest · Playwright
